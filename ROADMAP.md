# Roadmap

Current version: 1.0.0

This document lists planned directions for future versions. Items are not in strict priority order and may change as the project develops.

## Planned additions

**User bpy hooks**
Allow users to define custom operations by running their own Blender scripts within the pipeline environment, instead of being limited to built-in operation types.

**Generation lifecycle hooks**
Entry points for user-defined actions at different moments of the generation process, such as before or after a render, or before or after a full batch.

**Constraint system**
A constraint system for the randomized pipeline, so that sampled values can be restricted or related to each other rather than always being independent.

**Approximate thermal imaging**
A system to approximate thermal camera output for generated datasets.

## Contributing

If you want to help with any of the items above, open an issue first to discuss the approach before submitting a pull request.
